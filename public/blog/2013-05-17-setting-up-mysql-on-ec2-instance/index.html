Setting Up MySQL on EC2 Instance

Before setting up MySQL on EC2, you firstÂ need to install MySQL on your server.

See this post for setting up your web server on EC2 Instance.

  1. Make sure you give access to your MySQL database externally, by adding MySQL port 3306 to your Security Group. You can do this right in your Amazon Console Management, under Security Groups tab.

  2. Login to your instance through SSH & start mysql server. 

    sudo service mysqld start

  3. Set a password for root user. 

    /usr/bin/mysqladmin -u root password ‘yourpassword’

  4. Create a MySQL user.

Connect to your MySQL server using root user 

mysql -u root -p
CREATE USER ‘myuser’@‘localhost’ IDENTIFIED BY ‘yourpasswordhere’;
GRANT ALL PRIVILEGES ON . TO ‘myuser’@‘localhost’ WITH GRANT OPTION;
CREATE USER ‘myuser’@‘%’ IDENTIFIED BY ‘yourpasswordhere’;
GRANT ALL PRIVILEGES ON . TO ‘myuser’@‘%’ WITH GRANT OPTION;
